# Customer User Stories

## Authentication

### US-001

**As a customer**

I want to register an account

**So that**

I can purchase products from StyleHub.

---

### US-002

**As a customer**

I want to login to my account

**So that**

I can access my profile and orders.

---

### US-003

**As a customer**

I want to logout securely

**So that**

my account remains protected.

---

### US-004

**As a customer**

I want to reset my password

**So that**

I can regain access to my account.

---
